Python与Node.js

2019-12-08 PV:

基本定义

  • Python：一种面向对象的动态类型语言，起源于90年代早期。
  • Node.js：一个允许JavaScript前后端开发以及解决兼容问题的环境。

共同点

侧重语言

Python和Javascript都是脚本语言，都需要解释器来运行，都是动态类型，都支持自动内存管理,都可以调用eval（）来执行脚本等等脚本语言所共有的特性。

不同点

侧重环境

描述 Python Node.js
速度 编写快 执行快
门槛 语法简单、易调试 /
资源 框架文档丰富 文档滞后，无核心库和工具
社区 庞大积极 /
适用 原型、数据分析、渲染、爬虫、自动化脚本 内容、实时、社交、交易
案例 Instagram、C4D Paypal、ebay

应用领域

Python 是“简单”但“功能强大”的多功能编程语言”，具有高级文档和高级开发框架。它是大数据应用程序的首选语言，也适用于业务解决方案、教育平台、科学和医疗保健应用程序。

Javascript设计之初是一种客户端的脚本语言，主要应用于浏览器，它的语法主要借鉴了C；Node.js 是开发广告服务、游戏平台或论坛等应用的首选技术。它擅长处理具有大量同时连接或具有高速和密集 I/O 的应用程序的项目，以及生产力平台（如内容管理系统）、P2P 市场和电子商务平台等应用程序。Node 可用于不同类型的 Web 应用程序。

总结

编程语言和框架是为实现特定的项目目标而设计的，这是选择语言的主要标准。例如，某些技术更适合大数据应用程序（如Python和R），而其他技术更常用于构建大型桌面应用程序（如Java和C / ++ /＃）。

补充

  • 由于Node.js对实时的技术要求满足的比较好，这也是为什么在聊天机器人产品当中，会采用JavaScript语言去编写程序；

  • pip：python的一个包管理工具；

  • npm：node.js提供的一个包管理工具；

  • 前文分享的博客搭建，Hexo本质是基于Node.js环境的博客框架；当时的流程就是先安装Node.js环境，然后安装Node.js的包管理工具，再然后安装Hexo框架。
