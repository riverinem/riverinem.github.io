创新工作坊回顾

2019-12-01 PV:

前言

本篇是回顾年中参加的由普禾资本旗下研究院与XXY Innovation组织的「智能语音语义与图像技术应用创新工作坊」，工作坊围绕 声希科技的语音、语义及图像等核心技术，利用设计思维等相关工具和方法论，开展技术创新应用及商业模式的研讨。本次研讨以PM身份作为被邀对象，与业界技术专家、投资人、设计总监和教授等，在XXY Innovation创始人、著名设计学者辛向阳教授的指导下，参与技术产品化的讨论。

查询

基本查询：select * from

条件查询：select * from <表名> where <条件表达式>

  • <条件1> AND <条件2>
  • <条件1> OR <条件2>
  • NOT <条件>
  • (<条件1> OR <条件2>) AND <条件3>

投影查询：select <字段1>，<字段2>，<字段3> from <表名>

排序查询：select * from <表名> ORDER BY <字段1>，<字段2>

分页查询：LIMIT OFFSET 截取 第m条到n条的数据

聚合查询：

  • COUNT() 某列个数
  • SUM () 某列合值
  • AVG () 某列均值
  • MAX () 某列最大
  • MIN () 某列最小

多表查询：

  • SELECT * FROM <表1>, <表2> 返回的为两表乘积
  • SELECT <表1.旧字段 新字段>,<表2.旧字段 新字段> FROM <表1>, <表2> ; 返回多个某表某字段段数据，替换字段名
  • <表名/列名> +<空格>+ <替换名> 返回的数据以替换名呈现

连接查询：

  • FROM <表1> INNER JOIN <表2> 两张表都存在的记录
  • FROM <表1> LEFT OUTER JOIN <表2> 将表2的数据加到表1上
  • FROM <表1> RIGHT OUTER JOIN <表2> 将表1的数据加到表2上
  • FROM <表1> FULL OUTER JOIN <表2> 选出两表都存在的记录

增、删、改

增：INSERT INTO <表名> (字段1, 字段2, …) VALUES (值1, 值2, …)

删：DELETE FROM <表名> WHERE …UPDATE：更新已有记录

改：<表名> SET 字段1=值1, 字段2=值2, … WHERE …
