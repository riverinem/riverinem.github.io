Python简介


Python是一种解释型的、面向对象的、带有动态语义的高级程序设计语言；诞生于1991年，源自电视剧《Monty Python》，介于C和Shell之间，功能全面、易学易用、可扩展，特点是优雅、明确、简单。

Python应用于Web开发、GUI开发，与很多操作系统紧密结合，也可用于多媒体中。

Zen of Python：优美胜于丑陋，明了胜于晦涩，简洁胜于复杂。

Python运行方式

方式一：Shell方式

Shell是交互式的解释器，输入一行命令，解释器就解释运行出相应的结果。适用于代码段短。

方式二：文件方式

在python IDE环境中，创建一个以py为扩展名的文件，用python解释器在shell中运行结果。适用于代码段长。

推荐


jupyter是一种交互式计算和开发环境的笔记，ipython命令行比原生的python命令行更加友好和高效，还可以运行web版的界面，支持多语言，输出图形、音频、视频等功能。

Python环境配置与管理

pip安装与使用

pip 是 Python 包管理工具,该工具提供了对Python 包的查找、下载、安装、卸载的功能。

  1. 查看是否安装pip：pip --version

  2. 执行安装pip

    • curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
    • sudo python get-pip.py 运行安装脚本
    • pip install --upgrade pip 更新pip版本

  3. 指定python版本安装脚本sudo python3 get-pip.py 运行安装脚本。

关于pip和pip3的区别，实际上应该说是一样的，只是为了区别python2和python3之间的调用，避免冲突而进行的设定。当你安装了python3，使用pip和pip3都一样，如果只安装了python2，就不能使用pip3。

常用命令

  1. 获得帮助pip --help

  2. 升级pippip install -U pip

  3. 安装包

    • pip install SomePackage 最新版本
    • pip install SomePackage==1.0.4 指定版本
    • >pip install 'SomePackage>=1.0.4 最小版本

  4. 升级包pip install --upgrade SomePackage

  5. 卸载包pip uninstall SomePackage

  6. 搜索包pip search SomePackage

  7. 查看指定包信息pip show -f SomePackage

  8. 列出安装的包pip list

  9. 查看可升级的包pip list -o

Python版本管理工具：Pyenv

pyenv是Python版本管理工具。pyenv可以改变全局的Python版本，安装多个版本的Python，设置目录级别的Python版本，还能创建和管理vitual python enviroments。所有的设置都是用户级别的操作，不需要sudo命令。步骤如下：

  1. 安装homebrew

  2. 安装pyenv

    • brew update
    • brew install pyenv
    • brew upgrade pyenv 更新pyenv用
    • 输入echo 'eval "$(pyenv init -)"' >> ~/.zshrc 配置变量环境
    • 输入source ~/.zshrc，否则在切换python版本时应用不了（亲测）

  3. 安装python版本

    • 查看Pyenv可安装的Python版本pyenv install -l
    • 安装指定的python版本pyenv install 3.7.5
    • 安装完成后必须执行pyenv rehash （每当增删python版本或pip后使用）
    • 查看可使用的版本，前面带*表示当前使用的版本pyenv versions
    • 查看当前生效的版本pyenv version

  4. Python版本管理

    • 全局设置输入pyenv global <version> ，设置python版本为系统指定版本，设置全局的 Python 版本，通过将版本号写入 ~/.pyenv/version 文件的方式。
    • 本地设置首先cd <项目路径>，然后输入pyenv local <version> ，配置所在项目（目录）的python版本。
    • Shell设置输入pyenv shell <version> ，设置当前shell窗口使用的python版本为指定版本。退出shell后即失效；或者输入python shell --unset取消配置；这个版本的优先级比 local 和 global 都要高。

  5. Python版本修改生效查看which python 或者python --version ；

  6. Python卸载pyenv uninstall <version>

  7. 卸载Pyenv

    • rm -fr ~/.pyenv
    • 再删除.bashrc
    • export PATH=”~/.pyenv/bin:$PATH”
    • eval “$(pyenv init -)”
    • eval “$(pyenv virtualenv-init -)”

  8. 安装路径一般为：

    /Users/name/.pyenv/versions/3.7.5

配置Python虚拟环境：virtualenv

在同时开发多个应用程序时，不同项目需要的python版本不同，这种情况下，每个应用可能需要各自拥有一套“独立”的Python运行环境。virtualenv就是用来为一个应用创建一套“隔离”的Python运行环境。

常见问题

  1. 用Pyenv安装python版本遇到：BUILD FAILED(OS X 10.14.2 using python-build 20180424)，Inspect or clean up the working tree at …；使用如下语句安装：参考Pyenv安装Python多版本报错

  2. 用终端安装github上的requirements.txt清单，先用pyenv切换对应的python环境，然后输入pip install -r <txt路径>；另外还可以使用pip freeze > requirements.txt导出清单。

  3. 为什么通过pip安装的第三方库经常安装失败，出现time read out等问题，原因是安装源的问题，更换成国内的镜像源就会快很多，推荐阿里云、清华、豆瓣等国内源，速度提升明显。

Python输入与输出

输入：Python使用input函数实现驶入，input()返回的类型是字符型，如果需要整数型或者浮点型，需要用到int和float函数。

输出：Python使用print函数实现输出：Print(变量)、Print(字符串)。

Python风格

  • 注释：# ；
  • 续行符：/ （小、中、大和花括号系统可以判断，不需要加续行符）；
  • 一行多句：, ；
  • 缩进：用缩进表示语句级别关系，增加缩进表示开始，减少表示退出。

Python语法基础

  • 变量名：引用对象，标识对象。
  • 标识符：是指Python语言中作为变量名或其他对象名称的有效符号。
    • 首字符是字母或下划线；
    • 其余可以是字母、下划线或数字；
    • 大小写敏感；
  • 关键字：是Python语言的关键组成部分，不可随便作为其他对象的标识符。
  • 表达式：即用运算符连接各种类型数据的式子。
    • 运算符有优先顺序；
    • 表达式必须有运算结果；
  • 赋值：= ，变量第一次赋值，同时获得“类型“和“值”。
    • 增量赋值：m%5等于m=m%5
    • 链式赋值：x=2b=c+2
    • 多重赋值：
  • 语句：完整执行一个任务的一行逻辑代码。

Python标准数据类型

  • 整型1、2、3...n
  • 浮点型9.0
  • 布尔型true/false
  • 复数型实数+虚数
  • 序列类型：字符串单引号、双引号、三引号 、列表[] 、元组()
  • 映射类型：字典{a:1,b:2...z:26}

Python函数与模块

函数定义

可以看成类似于数学中的函数，可以完成特定功能的一段代码。

内建函数

  • 数值型内建函数：abs、bool、int、float；
  • 实用函数：input、help、open、len、range；

模块

包含函数、类等重复用到的其他代码块中，使用时需要用import 语句；比如引用math模块中的函数，需要输入import math ，而模块现实中就是一个像math.py的python文件。

包

一个有层次的文件目录结构，定义了一个由模块和子包组成的Python应用程序执行环境。

库

是一组具有相关功能的模块的集合。

小结

包=模块+库

库=模块1+模块2+…

模块=函数+类

Python语句

  • 条件语句与条件嵌套
    • if
    • else
    • elif
  • range语句：range(起始值包含 ,终值不包含,步长不为0)
  • 循环语句
    • while：当满足表达式时，执行循环语句；
    • for：遍历可迭代对象，如字符串、列、元组、字典和文件；
  • break语句与Continue语句
  • 自定义函数：def函数名 (参数 )：代码块；
    • 传递函数：函数可以像参数一样传递给另外一个函数；
    • lambda函数
  • 递归函数

Python数据获取

本地数据获取

  • 文件打开：
    • 参数1：路径（文件路径）必选
    • 参数2：模式（r-读，w-写）可选
    • 参数3：缓冲（默认-1，0代表不缓冲）可选
  • 文件读写：
    • f.read()、f.write()、f.readline()、f.writelines()
    • f.close()
    • f.seek()

mode和buffering为可选函数，默认值分别为r和-1；

网络数据获取

一般从网络上获取的数据可以分为：

  • 结构化数据：先有数据后有结构；
  • 非结构化数据：先有结构后有数据；
  • 异构数据：需要采用不同方式采集。

根据数据类型不同，爬取方式也有相应调整：

  • 结构化数据
    • 文本：正则表达式
    • html文件：正则表达式/xpath/bs4/css选择器
  • 非结构化数据
    • json文件：jasonPath
    • xml文件：正则表达式/Xpath/css选择器/
