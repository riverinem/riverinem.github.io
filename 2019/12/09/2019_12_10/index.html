2）分类模型指标概念

解释：假设下方表格灰色方块内数字为样本数据

预测值=1 预测值=0
真实值=1 True Positive(TP)5 False Negative(FN)2
真实值=0 False Positive (FP)4 True Negative(TN)4

准确率（Accuracy）= (TP+TN) / (TP+TN+FN+TN)= (5+4) / 15 =0.6

精准率（Precision）=TP/(TP+FP) = 5 / (5+4) =0.556

召回率（Recall）=TP/(TP+FN) =0.714

F1 = 2 * TP / (2 * TP+FP+FN) = 10 /（10+4+2）=0.625数值在0～1之间

3）回归模型指标概念

平均绝对误差（Mean Absolute Deviation）

平均绝对误差的计算方法是，将各个样本的绝对误差汇总，然后根据数据点数量求出平均误差。通过将模型的所有绝对值加起来，可以避免因预测值比真实值过高或过低而抵销误差，并能获得用于评估模型的整体误差指标。

均方误差（mean-square error, MSE）

均方误差=残差（预测值与真实值的差值）被求平方。对残差求平方的一些好处是，自动将所有误差转换为正数、注重较大的误差而不是较小的误差以及在微积分中是可微的（可让我们找到最小值和最大值）。

在模型预测中，模型可能出现的误差来自两个主要来源，即：因模型无法表示基本数据的复杂度而造成的偏差（bias），或者因模型对训练它所用的有限数据过度敏感而造成的方差（variance）。我们会对两者进行更详细的探讨。

偏差：如前所述，如果模型具有足够的数据，但因不够复杂而无法捕捉基本关系，则会出现偏差。这样一来，模型一直会系统地错误表示数据，从而导致预测精度低。这种现象叫做欠拟合（underfitting）；在此情况下，我们向模型提供多少数据并不重要，因为简单模型根本无法表示其中的基本关系，我们需要更复杂的模型。

方差：出现方差是正常的，但方差过高表明模型无法将其预测结果泛化到从中抽取训练样本的较大母体。对训练集高度敏感也称为过拟合（overfitting），而且通常出现在模型过于复杂或我们没有足够的数据支持它时。通常，可以利用更多数据进行训练，以降低模型预测结果的方差并提高精度。

监督学习

监管学习就是给定一堆样本，每个样本都有一组属性和一个类别，这些类别是事先确定的，那么通过学习得到一个分类器，这个分类器能够对新出现的对象给出正确的分类。这样的机器学习就被称之为监督学习。

机器学习算法

1）分类 Classification

对模型输入特征（feature），输出标签（label）；例如在网易云音乐中，根据用户常听的歌曲节奏和时间构建一个分类模型，当用户在某一个时刻打开APP时，然后判断哪些是该用户这个时段可能喜欢听的歌曲，并把这些歌曲打上标签推荐给用户。

2）回归 Regression

对模型输入特征（feature），依变量为离散输出、连续输出的标签（label）；

1⃣️直线回归：研究的是一个依变量与一个自变量之间的回归问题；

2⃣️多项式回归：研究一个因变量与一个或多个自变量间多项式的回归分析方法；

在畜禽、水产科学领域的许多实际问题中，影响依变量的自变量往往不止一个，而是多个，比如绵羊的产毛量这一变量同时受到绵羊体重、胸围、体长等多个变量的影响，因此需要进行一个依变量与多个自变量间的回归分析，即多元回归分析。

3⃣️参数回归：从最简单的f(x)=wx+b加入多个w，x变高阶，一般越复杂的模型，参数越多；

4⃣️K最近邻：给定一个训练数据集，对新的输入实例，在训练数据集中找到与该实例最邻近的K个实例， 这K个实例的多数属于某个类，就把该输入实例分类到这个类中。

3）决策树 Descision Tree

「定义」
决策树是一种十分常用的分类方法。他是一种监管学习。在机器学习中，决策树是一个预测模型，他代表的是对象属性与对象值之间的一种映射关系。

「优劣」

优点 速度快 准确高 处理连续和种类字段 适用于高维数据
缺点 样本不同，信息增益偏向多方 容易过拟合 忽略属性关系 /

「引申」

Entropy（熵），用以描述系统的凌乱程度；

在使用决策树时，较低的熵指向更有条理的数据，决策树将此用作事件分类方式。

「理念」

偏差和方差是两个模型误差中的两个来源，高偏差的模型会忽略数据，低偏差会使模型更加刻板，但在机器学习中希望算法是中庸的，具有适当的偏差达到一定泛化能力，敏锐而开放。
